# User Journeys & Конверсионная логика

Источник персон — `PRODUCT_BLUEPRINT §14` (5 аватаров) и `RESEARCH §15-16`.
Схлопнуто в 4 рабочие персоны + 4 сценария.

---

## Персоны (4)
1. **Корпоративный организатор** (B2B, ~70% выручки) — event-менеджер / HR.
   Хочет: быстро, надёжно, с КП для босса, без «сюрпризов».
2. **Невеста / Wedding-планер** (B2C, 25-30%) — эмоции, красота, «вау».
3. **Организатор частного события** (дни рождения, юбилеи, 10-15%) — уют, персонализация.
4. **Клиент «шеф у меня дома»** (premium, ~5%) — камерный ужин 2-12 чел, wine-pairing.

## Сценарии (пути)
```
[1] Корпоратив
Заход → /events/korporativ → выбор формата (фуршет/кофе-брейк/банкет)
→ /plan/calculator (гости+пакет+опции) → мгновенное КП (PDF)
→ /contact (или звонок менеджеру) → submit заявки → **/thank-you** (подтверждение) → менеджер звонит за 15 мин

[2] Свадьба
Заход → Hero (вау) → /events/svadba → галерея свадеб
→ /menu/furshet (рекомендованный премиум) → /plan/constructor (собрать своё)
→ не нашли меню? → «Соберём индивидуально» → заявка

[3] Частное событие
Заход → /events/chastnoe → пакеты (детский/BBQ/Новый год)
→ /menu/catalog (поштучно) → /plan/calculator → заявка

[4] Chef-at-home / sommelier
Заход → /events/chef-at-home → почасовая цена + опции (официант/сомелье)
→ форма «персональное предложение» → шеф связывается за 2 часа
```

## Конверсионная логика по блокам (шаблон)
| Блок | Цель | Что НЕ делать |
|---|---|---|
| Hero | «хочу так же» + 1 CTA | не грузить текстом, не 5 кнопок |
| Trust bar | снять риск «а они надёжные?» | не просто логотипы — с подписью «Эрмитаж, Мариинский» 🟡 (клиенты НЕ подтверждены вебом — публиковать только как 🟡 пример / после письменной сверки с заказчиком; см. FACTCHECK_REPORT_2026-07-20) |
| Event selector | направить по пути | не заставлять гадать, куда жать |
| Menu showcase | возбудить аппетит | не список блюд «как в меню ресторана» |
| Gallery | «я хочу такое же» | не разрозненные фото, а по типам событий |
| Reviews | доверие | не только звёзды — реальные кейсы с именем/событием |
| Calculator | мгновенная оценка бюджета | не 15-минутный квиз (у конкурентов тормозит) |
| Constructor | персонализация без перегруза | не 100 блюд с галочками — мастер 6 шагов |
| CTA / Contact | заявка или звонок | всегда видимый телефон + WA + форма |
| NewsletterSignup | подписка на новости (152-ФЗ) | вести на /api/newsletter, НЕ мёртвая ссылка |
| RatingBadge (Google) | внешний реальный рейтинг | ссылка на профиль Google Business, НЕ фейковый AggregateRating |
| StickyMobileCTA | заявка в 1 тап с любой глубины | [benchmark pxlpeak «Thirsty Click»] не прятать CTA на mobile |
| AvailabilityCalendar | доверие + дефицит | не «мы всегда свободны» — показать реальную занятость |

## Главный принцип (из blueprint §1.4)
**5 шагов до конверсии** (а не 8-10 у конкурентов). Каждый лишний шаг ≈ −20% конверсии.
Сайт = «предварительная диагностика» заказа: когда клиент отправляет заявку,
менеджер получает контекст (тип, гости, меню, аллергены, бюджет) → звонит как эксперт, не оператор.

## Служебные / инфраструктурные страницы (Волна 2А)
- **/thank-you** — целевая страница после успешного submit `ContactForm` (и пост-калькулятор/конструктор → `/api/contact`).
  Контекст: «Заявка принята, перезвоним ≤15 мин», CTA «К спланированному событию» → `/plan`. НЕ в навигации, не индексируется (`noindex`).
- **/404** — кастомный not-found: живой тон («Этой страницы нет, но кейтеринг — есть») + быстрые ссылки (Главная / Меню / События / Контакты). Глобальный `notFound()` → эта страница.
- **/reviews** — ПОЛНАЯ страница отзывов (расширение блока `ReviewCard`/`ReviewList`, `04_BLOCKS`). Фильтр по типу
  события (`?type=wedding|corporate|private|chef-at-home`) + связь с `TestimonialsCarousel` на главной. Реальный Google-рейтинг — `RatingBadge`.
- **/delivery** — зоны доставки + карта: КАД включён (0 ₽), надбавки вне КАД (таблица зон/районов ЛО). Связь с `TrustProof` («КАД включена»).
- **/certificates** — сертификаты и безопасность (152-ФЗ, ТР ТС 021/2011, Декларация соответствия, Роспотребнадзор, аккредитация кухни). Закрывает битую футер-ссылку бага C из `17_IA_CRITIC`.
