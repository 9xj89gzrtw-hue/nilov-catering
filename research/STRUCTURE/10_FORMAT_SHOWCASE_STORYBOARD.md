# FormatShowcase — текстовый раскадровка (light, editorial)

**Цель:** после Hero показать 3 формата обслуживания как «живые» карточки, ведущие в меню.
**Блок:** `FormatShowcase` (из `04_BLOCKS.md`). **Токены:** `06_TOKENS.md`.

---

## Визуальный слой
1. **Фон секции:** `--color-bg` (ivory), сверху тонкий разделитель `--color-line`.
2. **SectionHeading (слева):** overline «Что вы задумали?» (gold-text `#8A6D3B`) + H2 Cormorant 44px «Три формата обслуживания».
3. **3 карточки** (grid `repeat(3, 1fr)`, desktop / stack mobile), каждая:
   - фото-обложка (16:10) с photo-alive hover (Ken Burns zoom; [Волна 12, B5] 3D-tilt ±4° ТОЛЬКО на фото-обложке, тело карточки — flat, чтобы текст/CTA не «плыли»),
   - название формата (Cormorant 28px),
   - одна строка сути (Inter 14px ink-soft),
   - цена «от X ₽ / гость» (JetBrains Mono, **ink** `#1C1815`; [Волна 12, B2] золото только underline под ценой, НЕ цвет цифр),
   - микро-бейдж (напр. «Хит свадеб»),
   - CTA-строка «Смотреть меню →» (ghost, hover gold-text `#8A6D3B`).

## Motion поведение
| Элемент | Анимация | Тайминг |
|---|---|---|
| SectionHeading | fade-up (Reveal wrapper) | on scroll-in, 400ms |
| Карточки | staggered fade-up (80ms between) | on scroll-in |
| Фото (hover) | Ken Burns scale 1→1.08 (6s) + 3D-tilt ±4° ТОЛЬКО на фото-обложке (desktop, `pointer:fine`, in-view) | hover [B5] |
| Карточка (hover) | translateY(-6px) + `--shadow-hover` | hover, 200ms ease-out-expo |
| CTA (hover) | цвет ghost → gold-text `#8A6D3B` (4.54:1 AA) + стрелка сдвиг +4px | hover |
| Mobile | фото tap → lightbox (без tilt) | tap |

## Что делает блок «trustworthy» (чек-лист)
- [ ] Фото — кинематографичные кадры еды/сервиса, НЕ плоские «блюдо на белом».
- [ ] Цена в JetBrains Mono **ink** `#1C1815` (16.5:1 AAA), золото только underline (технично = прозрачно = доверие) [Волна 14, B2].
- [ ] Только одна CTA-строка на карточку, без кнопочного мусора.
- [ ] Hover живой, но сдержанный (не радужный).
- [ ] Карточки на ivory, НЕ «плавающие» тяжёлые плитки.
- [ ] Ведут в `/menu/*`, а не куда попало.

## Критерий приёмки блока
Клик по «Смотреть меню» ведёт на соответствующий раздел; на hover фото оживает;
на scroll секция плавно въезжает. Никаких «заглушек» с серым фоном.
