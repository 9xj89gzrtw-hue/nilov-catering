1. **Балл ≥8 требует ДОКАЗАТЕЛЬСТВА** (собранный код / Lighthouse / axe / скриншот).
   Ни один блок не развёрнут → **потолок 7** по `18` §5. Ни один блок ≠ PASS.
2. **a11y / Perf — НЕ держим на ≤5**, если в spec ЕСТЬ стратегия. По `18` §5 и задаче:
   - spec содержит a11y-стратегию (aria/alt/reduced-motion) → **a11y 6–7** (не 9–10 без axe).
   - spec содержит LCP/INP/CLS-бюджет + стратегию → **Perf 6–7**.
   - блок полный (пропсы/копирайт/данные/ассеты/a11y/perf/responsive) → остальные оси **7–8**
     при наличии цитат бенчмарка (`16`).
3. Проверяем ТОЛЬКО цитатами из `04`/`16`/`07`. Заявления автора без цитаты не считаются.
