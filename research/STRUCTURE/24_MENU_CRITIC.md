## 1. ПРОДАЖНОСТЬ на 2026 — 5/10

**Почему не выше (devil's advocate):**
- Черновик 23 **откатился в generic**. Старое меню `NILOV_UNIFIED_MENU.md` (историческая база, НЕ канон) было *удивительно тренд-выровнено*: бао, поке, будда-боулы, молекулярная подача, шутеры-супы, show-cooking, десерт-станции в Люкс, сезонная ротация 20%. 23-й скелетон это **всё выкинул** и свёлся к «закуски/горячее/десерты/напитки/веган/детское». Это регресс в дифференциации = хуже продаётся, потому что не цепляет.
- В 23 **нет ни одного из 6 якорей-2026** (см. Johnny Burke / Upohar / BREADLESS / KanteenSF):
  1. Seasonal sourcing как базис — только упомянуто косвенно;
  2. **Global small plates** (бао, тако, поке) — были в старом, исчезли в 23;
  3. **Interactive / grazing stations** — есть как формат «Шоу-кулинария», но НЕ как меню-категория с блюдами;
  4. **Plant-based mains, которые стоят сами по себе** (vegan + GF) — в 23 только абстрактно «Веган»;
  5. **Signature cocktails / моктейли** вместо «открытого бара» — в 23 только «Напитки (безалк/алк/комбуча/кофе)», без программы;
  6. **Dessert stations / mini-dessert shooters** вместо одного торта — в 23 только «Десерты (в т.ч. мини-десерты порционно)», без станций.
- Категории 23 **не привязаны к трендам** «mini-desserts», «comfort-global», «zero-waste veg-centric» — значит меню будет выглядеть как «ещё один фуршет», а не «самый популярный на 2026».

**Что засчитано в плюс:** форматы (фуршет/банкет/кофе-брейк/корпоратив/шоу/chef-at-home) и тарифные сетки из `07` — рыночно верны; категорий достаточно по количеству. Но «структура есть, товара нет» = продажность = средняя.

## 2. ДОСТАТОЧНОСТЬ ПОЗИЦИЙ — 2/10

- В черновике **0 конкретных блюд**. Критерий «хватает ли выбора по каждому формату/категории» = **НЕТ** по определению.
- Бенчмарк реальности: Jacks.ru держит **30+ категорий** с десятками SKU в каждой; даже старое `NILOV_UNIFIED` имело полный каталог (~60–80 позиций) + 6 станций + детское + бранчи.
- Минимумы по формату из `07_CALCULATOR_SPEC` (канон): фуршет ≥8 (4 холодн.+2 горяч.+2 десерт), банкет ≥6 (закуска+суп+2 горячих+гарнир+десерт), кофе-брейк ≥5 (выпечка+напитки+сэндвичи+фрукты+кофе-станция), mobile-фуршет ≥6. 23-й **не фиксирует эти минимумы** внутри себя → риск недобора при наполнении.
- Балл 2, а не 1: структура категорий задана, «дерево» есть, осталось насадить листья.

## 3. ЗАКРЫТИЕ ПОТРЕБНОСТЕЙ — 4/10

| Потребность | Статус в 23 | Оценка |
|---|---|---|
| Сценарии: фуршет / банкет / кофе-брейк / корпоратив / свадьба / частное / chef-at-home | Названы как форматы | ✅ намерение есть |
| Диеты: веган / безглютен / халяль / детское | Названы категориями | ⚠️ без конкретных блюд и без **сертификации халяль** (в старом — «по запросу», этого мало для продажи) |

| Бюджеты (эконом‑премиум) | Тарифы в `07` (4/3/4) | ✅ но нет «пола» для сверх-бюджетных (деловой ланч 700 ₽, кофе 950 ₽ — есть в старом, не в 23) |

**Вывод:** закрытие потребностей **задекларировано, но не реализовано**. До >8 нужно: по каждому сценарию — готовый набор блюд; по каждой диете — счётное число блюд (см. эталон); халяль — либо убрать из обещаний, либо дать сертифицированную линию; 14 аллергенов — привязать к схеме данных каталога.

## 4. ЭТАЛОННЫЙ СПИСОК ПОЗИЦИЙ (что реально продаётся в 2026)

> Собрано из web-trends 2026 + базы продаж СПб (Jacks/CaterMe/старое NiloV). Это и есть то, чем Author должен наполнить `23`. Структура — «5+2» (OkCatering): 5 основных категорий + 2 доп. блока; баланс выхода 40% закуски / 30% горячее / 20% десерты / 10% напитки (art-active).

### A. ТРЕНД-ЯКОРЯ 2026 (обязательно добавить в 23, их сейчас НЕТ)
1. **Глобальные мини-плейты (global small plates):** бао с томлёной свининой; бао с уткой и хойсином; мини-тако (рыба/цьветная капуста); тайские мясные шарики с чили; jerk-курица на плантане; поке с лососем и авокадо; будда-боул с тофу.
2. **Plant-based mains «сами по себе» (веган + GF):** цуккини-пурсе с кокосовым карри; стейк из цветной капусты с тапенадой; овощное рагу; карпаччо из свёклы с дымком; бургер из грибов/бобов.
3. **Интерактивные станции (как меню-категория, не только формат):** телятина на камне; целиком запечённый лосось; тако-станция; ризотто в колесе пармезана; раклет-бар; строганина из северных рыб.
4. **Grazing table / чаркутери:** мясное плато (буженина/ростбиф/индейка); сырная тарелка (камамбер/маасдам/пармезан + мёд/орехи); буррата + томаты.
5. **Десерт-станции и мини-шутеры (вместо одного торта):** макаронс; донат-стена; чизкейк-шутеры; молочно-печенье-шутеры; шоколадный мусс (веган) в креманках; веррины чиа-пудинг с манго; брауни; маффины; фруктовая тарелка.
6. **Программа напитков (signature, не просто «бар»):** моктейли (облепиховый чай с имбирём, клюквенный морс с мятой, кедровый раф); авторские лимонады (эстрагон/ягоды); комбуча; кофе-станция; welcome-drink (игристое + канапе).

### B. ЭТАЛОННЫЙ КАТАЛОГ ПО ФОРМАТАМ (минимумы из `07` соблюдены)

**ФУРШЕТ (от 20, ≥8 позиций)** — холодные 4 / горячие 2 / десерт 2:
- Холодные: канапе с лососем; канапе с красной икрой; тарталетка с оливье; брускетта с томатами/базиликом; канапе «Капрезе» с песто; рулетик из лаваша с курицей; **бао/тако (тренд)**; мясное плато.
- Горячие: мини-бургер (свинина/курица); куриные якитори; овощи гриль; **цапф-капуста-стейк (vegan GF)**.
- Десерт: шоколадный брауни; веррин чиа-пудинг; **макаронс-шутер (тренд)**.
- Напитки: лимонад/морс; чай/кофе; минералка.

**БАНКЕТ (от 15, ≥6)** — закуска + суп + 2 горячих + гарнир + десерт:
- Суп: борщ / окрошка (сезон) / крем-суп из белых грибов.
- Закуски: мясное плато; салат «Цезарь»; салат с лососем и авокадо.
- Горячие: бефстроганов / медальоны из вырезки; форель с картофельным муссом; **стейк из цветной капусты (vegan GF)**.
- Гарниры: пюре / запечённые овощи / картофель с розмарином.
- Десерт: тирамису / яблочный тарт / чизкейк-шутер.

**КОФЕ-БРЕЙК (от 10, ≥5)** — выпечка + напитки + сэндвичи + фрукты + кофе-станция:
- Выпечка: круассан, эклер, маффин, макаронс.
- Сэндвичи/канапе: канапе с лососем; мини-сэндвич.
- Фруктовая тарелка.
- Кофе-станция + чай; моктейли/лимонады (тренд).

**КОРПОРАТИВ / БИЗНЕС-ЛАНЧ (от 15, пол/эконом):** суп + салат + горячее с гарниром + хлеб + напиток; гастробоксы (Бизнес-ланч / Премиум).

**СВАДЬБА:** банкет-Премиум/VIP + десерт-станция (вместо торта) + welcome-drink + флористика (бесплатно в старом — сохранить).

**ЧАСТНОЕ / Chef-at-home:** адаптация банкета под посадку дома; show-cooking на площадке.

**ШОУ-КУЛИНАРИЯ (станции):** см. п. A.3.

**СПЕЦПАКЕТЫ (проверенные хиты старого, оставить):** Летний BBQ (2200 ₽), Детский праздник (1200 ₽), Новогодний корпоратив (1970 ₽), Корпоративный обед (700 ₽).

**ДЕТСКОЕ МЕНЮ (отдельный блок, как в старом):** мини-бургер; наггетсы; мини-пирца; паста с сыром; фруктовые канапе; панкейки; капкейки; молочный коктейль; фруктовое ассорти.

**БРАНЧИ/ЗАВТРАКИ:** круассан с ветчиной; омлет; сырники; греческий йогурт с гранолой; свежевыжатый сок; латте.

**ВЕГАН/БЕЗГЛЮТЕН (явные линии, не «просто категория»):**
- Веган: будда-боул с тофу; цветная капуста-стейк; карпаччо из свёклы; овощное рагу; фруктовая тарелка; шоколадный мусс (авокадо).
- Безглютена: поке; боул с курицей и киноа; цветная капуста-стейк; салаты без гренок; мясное/сырное плато; фрукты.
- Халяль: **требует сертификации** — либо дать сертифицированную линию (баранина/птица по халяль), либо убрать из обещаний 23 (сейчас «халяль» назван, но реализации нет → риск претензий).

## 5. BUG-LIST ДЛЯ AUTHOR'А (что поправить в 23_MENU_STRATEGY.md)

**КРИТ (блокируют >8/10):**
- **B1. Пустой список позиций.** Заполнить `23` реальными блюдами из раздела 4 выше. Скелетон без блюд = 2/10 по достаточности. Цель: каталог 60–90 SKU, сбалансированный 40/30/20/10.
- **B2. Нет тренд-категорий 2026.** Добавить в 23 явные блоки: *Global small plates* (бао/тако/поке), *Plant-based mains* (vegan+GF), *Interactive stations* (как меню-категория), *Grazing table*, *Dessert stations & mini-shooters*, *Signature drinks/mocktails*. Без них меню «как у всех» = не «самое популярное 2026».
- **B3. Не зафиксированы минимумы позиций.** Прямо в 23 прописать: фуршет ≥8 (4 хол.+2 гор.+2 дес.), банкет ≥6, кофе-брейк ≥5, mobile ≥6 — чтобы наполнение не «недобрало» (канон из `07`).

**СРЕДН (повышают продажность/закрытие):**
- **B4. Халяль — честно.** Либо сертифицированная линия (баранина/птица), либо убрать из обещаний 23. «По запросу без сертификата» для продающего меню недостаточно.
- **B5. Диеты = счётные блюда, не ярлыки.** В 23 для каждой диеты дать N конкретных блюд (веган ≥6, GF ≥8, детское ≥8). Сейчас — только названия категорий.
- **B6. 14 аллергенов привязать к данным.** В 23 явно: «каждое блюдо каталога несёт теги 14 аллергенов ТР ТС + перекрёстная контаминация (*)»; это питает `AllergenFilter` из `07` и `TrustProof`.
- **B7. Сезонная ротация 20%.** Перенести из старого: лето/осень/зима/весна — что добавить/убрать. 23 об этом молчит, хотя это сильный продающий сигнал («сезонное = свежее/дешевле»).

**МЕЛКИЕ (полировка):**
- **B8. Бюджетный «пол».** Добавить в 23: деловой ланч 700 ₽ и кофе-брейк 950 ₽ (есть в старом/07) — чтобы закрывать низкобюджетные заказы, а не только 2450+.
- **B9. Не терять сильное старого.** Бао, поке, будда-боулы, молекулярная подача, шутеры-супы из `NILOV_UNIFIED` — тренд-выровнены; вернуть в 23 как «проверенные хиты», не изобретать заново.
- **B10. Show-cooking — конкретный прайс-лист станций** (телятина на камне / лосось / тако / ризотто / раклет / строганина) перенести из старого в 23, чтобы блок «Шоу-кулинария» был не пустым.
- **B11. Детское меню и бранчи — отдельными блоками** (как в старом), а не свалены в «Детское меню» одной строкой.
