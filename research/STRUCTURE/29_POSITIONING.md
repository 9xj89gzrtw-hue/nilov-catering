# 29 — POSITIONING BRIEF (важное уточнение пользователя, 2026-07-19)

**Точная формулировка пользователя:** «всё равно это должен быть лучший в мире сайт
кейтеринговой компании, просто не премиум».

**Что это значит (критично — не путать):**
- **МИРОВОЙ УРОВЕНЬ ИСПОЛНЕНИЯ** (оставить): топовый UX, Drinkit «живое фото»,
  безупречная типографика/анимации, идеальные CWV (LCP<1.2s), светлая airy ДНК (`01`).
  Это и есть «лучший в мире». НЕ упрощать визуал.
- **НЕ ПРЕМИУМ ПО ПОЗИЦИОНИРОВАНИЮ** (смягчить): средний сегмент, не VIP/элитно/
  эксклюзив/«для избранных». Премиум-мероприятия ВОЗМОЖНЫ, но НЕ в первую очередь.
  В среднем сегменте — опыт заказчика и легче работать.

**Различие (для исполнителя):**
- Глянец ДНК (light/airy/glossy) = КАЧЕСТВО исполнения → СОХРАНИТЬ.
- Премиум-ЛЕКСЕМЫ («VIP», «элитно», «эксклюзив», «от 100 000₽», «избранным») =
  СНОБИЗМ → УБРАТЬ.
- Заменить на: «под ключ для любого бюджета», «реальные цены», «доступно»,
  «делаем круто для любого события», «без переплат».

**Риск (по вебу):** skyfielddigital 2026 — real photos > glossy stock; ridgewells —
restraint как сигнал. Средний сегмент доверяет «реальным событиям/честным ценам»,
а не «элитности». Глянец + премиум-лексемы = риск отпугнуть. Глянец + тон «для всех» = ок.

**Что делать исполнителю (когда вернётся TrustSegCritic):**
1. В `01_VISUAL_DNA.md` — НЕ менять палитру/типографику/анимации (мировой уровень).
   Только добавить раздел «Позиционирование: средний сегмент, не премиум».
2. В `04_BLOCKS.md` — убрать премиум-лексемы из CTA/копирайта (Hero sub, TrustProof,
   PackageGrid лейблы), добавить «доступно/под ключ/реальные цены». AwardsStrip —
   оставить как соц-доказательство, но убрать «элитность».
3. В `23_MENU_STRATEGY.md` — цены помечены pending; тон меню «для любого события».
4. НЕ снижать визуальное качество ДНК ни в коем случае.

## БЮДЖЕТНЫЙ СТЕК (ЖЁСТКОЕ ТРЕБОВАНИЕ, 2026-07-19)

**Требование пользователя:** заказчик платит ТОЛЬКО за хостинг и домен. Всё остальное —
бесплатные решения, доступные в РФ. Никаких платных подписок (Vimeo/Storyblok/Typeform/платный CMS).

**Обязательный бесплатный РФ-стек:**
- **Хостинг:** VPS Timeweb/Selectel (РФ) ~500–2000₽/мес. Next.js SSR/ISR на нём.
- **Домен:** .ru/.рф ~200–500₽/год. SSL Let's Encrypt — бесплатно.
- **CMS (админка, бесплатно):**
  - Вариант А: **Sanity free tier** (облако, бесплатно, 20 seats) — контент меню/текстов/
    отзывов. Данные контента НЕ персональные → допустимо вне РФ.
  - Вариант Б (если параноя по 152-ФЗ): **Strapi self-host** (open-source, бесплатно)
    на том же VPS — всё в РФ. Больше настройки, но $0.
  - ❌ Storyblok ($99/мес) — ЗАПРЕЩЁН.
- **Видео (бесплатно, РФ):**
  - **Rutube** (бесплатно, РФ-резидент, доступен в РФ) — эмбед как Vimeo.
  - ИЛИ self-host MP4 на том же VPS (если трафик небольшой) — $0.
  - ❌ Vimeo ($20–70/мес, в РФ нестабилен/РКН) — ЗАПРЕЩЁН.
- **Соц-ленты (решение пользователя, 2026-07-19):** Instagram/WhatsApp/Telegram — **ОСНОВНЫЕ
  каналы заказчика** (круглосуточные сторисы с мероприятиями = живой UGC-доверия).
  - **Instagram КЕПТ** (не удалять) — аудитория сидит через VPN. Fail-soft: если эмбед не
    грузится (нет VPN) → ссылка «Мы в Instagram» + VK-лента как всегда-доступный fallback.
  - **WhatsApp / Telegram** — вынести в КОНТАКТЫ и sticky-CTA как первичные каналы связи
    (не только иконки в футере). Круглосуточный отклик.
  - ❌ НЕ удалять ни один из трёх (решение отменено — ранее планировал IG→VK).
  - VK — опциональный fallback (в РФ всегда доступен), НЕ замена.
- **Формы/заявки (бесплатно):**
  - Свой API route (Next.js) на том же VPS → пишет в **РФ-БД** (Postgres/SQLite на VPS).
  - Перс. данные (имя/телефон) — ТОЛЬКО на РФ-сервере (152-ФЗ локализация).
  - ❌ Typeform/платные формы — ЗАПРЕЩЕНЫ.
- **ERID-маркировка:** уточнить у LegalCritic; если обязательна — через аккредитованного
  (есть бесплатные/льготные пути) или сайт-каталог может не требовать. Не платить
  коммерческие аккаунты без необходимости.

**Итог расходов заказчика: домен (~300₽/год) + VPS (~500–2000₽/мес). Всё остальное $0.**
Это и есть «только хостинг и домен».

**ЗАПРЕТ в структуре (исполнитель обязан соблюсти):**
- ❌ Vimeo → ✅ Rutube / self-host.
- ❌ Storyblok → ✅ Sanity free / Strapi self-host.
- ❌ Платные формы/сервисы → ✅ свой API + РФ-БД.
- ❌ Любые monthly SaaS-подписки для базовой работы сайта.
