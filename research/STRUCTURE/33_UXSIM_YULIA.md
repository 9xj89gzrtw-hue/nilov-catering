**Роль:** Маркетолог №14 (`32_UX_PERSONAS`). Готовит тендер/презентацию заказчика, нужны материалы NiloV.
**Сценарий:** «Мне для тендера нужны фото реализованных проектов NiloV, логотип (для согласования/размещения), сертификаты (Роспотребнадзор/ТР ТС/Халяль) и кейсы — всё скачиваемое, чтобы вставить в презентацию. Отдельная страница «Для партнёров/Пресса» была бы идеальна.»
**Тон:** критик-персонаж, занижение. Балл ожидаемо ≤8.
**Источники:** `04_BLOCKS.md`, `02_IA.md`, `07_CALCULATOR_SPEC.md`, `08_CONSTRUCTOR_SPEC.md`, `32_UX_PERSONAS.md`, эталон `33_UXSIM_NINA.md` (сертификаты/документы).

**ФОКУС ПРОВЕРКИ (гипотезы разрывов):** (а) доступны ли сертификаты/кейсы для скачивания (медиа-кит), (б) разрешение на использование логотипа NiloV, (в) примеры проектов с фото, (г) страница «Пресса/Медиа/Для партнёров».

## ПРОИГРЫШ СЦЕНАРИЯ (от лица Юлии)

1. **Заход — ищу «Пресса/Медиа/Партнёрам».** `02_IA.md`: `/why-us`, `/team`, `/reviews`, `/blog`, `/certificates` — есть. НО нет `/press`, `/media-kit`, `/partners`. В навигации (`04`) — тоже нет.
2. **Сертификаты.** `/certificates` (волна 2А) существует — блок `CertBlock` с 152-ФЗ, ТР ТС 021/2011, декларацией. Но **скачать PDF сертификата нельзя** — это страница-просмотр, не медиа-кит. Юлия не может вставить сертификат в тендер как файл. `alt`/`download` отсутствуют.
3. **Логотип NiloV.** Нигде нет «Скачать логотип» (SVG/PNG + гайдлайн использования). Для тендера Юлии нужен официальный логотип + правила размещения (можно ли на слайде, цвета). Спека молчит.
4. **Фото проектов.** `GalleryTeaser`/`GalleryMasonry` (`04`) — галерея есть, НО фото НЕ скачиваемы (нет «download high-res»), и нет привязки «это проект для Эрмитажа/корпоратива X» — просто красивые кадры без контекста кейса. Юлии нужны **кейсы с описанием** (кто/что/сколько гостей), а не анонимная еда.
5. **Кейсы.** `TestimonialsCarousel` (видео-отзывы) есть, но это отзывы, не структурированные кейсы «NiloV сделал корпоратив на 500 чел для X — результат». `BlogPost.rubric` = case/tip/guide (волна 10 добавила guide) — кейсы есть как статьи, но НЕ в скачиваемом медиа-ките.
6. **Контакт для партнёров.** Нет выделенного «Для партнёров/прессы» контакта (почта pr@). Только общий `/contact`.

## BUG-LIST (что исправить для ≥9)

- **B-YULIA-1 (критично, медиа-кит).** Нет страницы/раздела «Медиа-кит / Для партнёров / Пресса». *Требование:* роут `/media-kit` (или `/press`) + блок `MediaKit` — агрегатор скачиваемых материалов (логотипы SVG/PNG, гайдлайн бренда, фото high-res, сертификаты PDF, кейсы).
- **B-YULIA-2 (критично, скачивание сертификатов).** `/certificates` — только просмотр. *Требование:* кнопка «Скачать PDF» на каждом сертификате (Роспотребнадзор/ТР ТС/Халяль/аккредитация) + архив всех сертификатов zip.
- **B-YULIA-3 (критично, логотип + гайдлайн).** Нет скачиваемого логотипа и правил использования. *Требование:* в `/media-kit` — логотип (цветной/ч/б, SVG/PNG) + `BrandGuidelines` (где можно размещать, цвета, отступы).
- **B-YULIA-4.** Фото галереи не скачиваемы и без контекста. *Требование:* `GalleryMasonry` несёт `downloadHighRes` + подпись-кейс («Корпоратив, 200 гостей, Эрмитаж»); кураторская подборка «Для тендера» в медиа-ките.
- **B-YULIA-5 (критично, кейсы как документы).** Есть отзывы/блог-case, но нет скачиваемых кейсов. *Требование:* `BlogPost.rubric:'case'` дублируется в медиа-кит как PDF-кейс (кто/что/сколько/результат).
- **B-YULIA-6.** Нет контакта для партнёров/прессы. *Требование:* в `/media-kit` и футере — выделенный `pr@nilov-catering.ru` (или форма «Для партнёров»).
- **B-YULIA-7.** `02_IA.md` не знает медиа-кит роута. *Требование:* дописать `/media-kit` в sitemap.
- **B-YULIA-8.** VideoClip/отзывы не несут download-постер. *Требование:* для тендерных вставок — скачиваемый постер-кадр видео-отзыва (с согласия клиента).
- **B-YULIA-9 (alt-связь с Анной A7).** Фото в медиа-ките ОБЯЗАНЫ нести осмысленный `alt` (см. волну 9/A7) — иначе для слабого зрения/скринридера Юлии неясно, что на фото.

## ВЕРДИКТ (коротко)
Сайт содержит сертификаты и галерею, НО они «закрыты» от выгрузки: нет медиа-кита, скачивания логотипа/сертификатов/фото-кейсов, страницы для партнёров. Юлия не может собрать тендерную папку из сайта. Главный разрыв — **отсутствие концепции downloadable media-kit**. Балл **5/10**.
